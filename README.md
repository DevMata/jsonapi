# BLOG JSON API

## Start

## blog endpoint

```
/GET /blogs returns blogs from most recent to last
/GET /blogs/1 returns blog with id 1

/POST /blogs create a new post, title and content are required, returns created post

/PUT /blogs/1 update post with id 1, title and content are required, returns updated post

/DELETE /blogs/1 deletes post with id 1
```
